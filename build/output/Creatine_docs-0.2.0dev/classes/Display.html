setSmoothingEnabled

(
  • value
)

Defined in setSmoothingEnabled:288

Sets the Image Smoothing property on the given context. Set to false to disable image smoothing. By default browsers have image smoothing enabled, which isn't always what you visually want, especially when using pixel art in a game. Note that this sets the property on the context itself, so that any image drawn to the context will be affected. This sets the property across all current browsers but support is patchy on earlier browsers, especially on mobile.

Parameters:

  • value Boolean
    • If set to true it will enable image smoothing, 
       false will disable it.

scaleMode

Constant

Defined in scaleMode:127

The scale mode of the canvas. Set this to use the automatic resizing. Accepted modes are:

  • creatine.FIT: scale the canvas to fit the available space, without exceeding it and respecting the aspect ratio.
  • creatine.STRETCH: scale the canvas to fit the available space, without exceeding it but does not respect the aspect ratio.
  • creatine.NOSCALE: (default) keeps the original size of the canvas.
