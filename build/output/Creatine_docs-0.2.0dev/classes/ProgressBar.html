ProgressBar Class

Defined in: ProgressBar:35
Module: Creatine

ProgressBar is a progress bar that handle continuous values. It is divided into three main types, which can be specified by colorsOrImg parameter:

  • ColorBar: by passing a string representing a color (e.g., '#0f3' or 'red'), the progress bar will be filled using this color.

  • GradientBar: by passing a list of colors (e.g., ['red', 'blue']), the progress bar will be filled using a gradient passing through all provided colors.

  • ImageBar: by passing a Bitmap or Image object, the progress bar will be filled with using the provided image.

A ProgressBar can filled in 4 different directions, by using the parameter direction: LEFT_TO_RIGHT, RIGHT_TO_LEFT, TOP_TO_BOTTOM, BOTTOM_TO_TOP.

Example

// Create the Director
var progress = new creatine.ProgressBar(
    'green',                     // The fill color
    'black',                     // The background color
    creatine.LEFT_TO_RIGHT       // The director
    100,                         // The bar width
    20,                          // The bar height
    0,                           // The minimum value
    100                          // The maximum value
)
