FadeOut Class

Defined in: FadeOut:37
Module: Transitions
Parent Module: Creatine

The FadeOut is a transition effect that fades out the current scene.

Example

director.replace(
    new MyScene(),
    new creatine.transitions.FadeOut(
        createjs.Ease.bounceOut, 
        400
    )
)
