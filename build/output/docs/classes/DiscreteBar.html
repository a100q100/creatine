DiscreteBar Class

Defined in: DiscreteBar:35
Module: Creatine

DiscreteBar is a progress bar that only handle discrete values. This is
a simple and specific implementation of a progress bar, for a more general version, consult the ProgressBar.

It can filled in 4 different directions, by using the parameter direction: LEFT_TO_RIGHT, RIGHT_TO_LEFT, TOP_TO_BOTTOM, BOTTOM_TO_TOP.

Example

// Create the Director
var progress = new creatine.DiscreteBar(
    image,                   // The image used to fill.
    5,                       // The space between one image and other.
    creatine.LEFT_TO_RIGHT   // The director.
)
